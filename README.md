# Random Quote Generator

**Автор:** Ильяна Кертбиева

## Описание

Приложение для генерации случайных цитат с графическим интерфейсом. Позволяет:
* генерировать случайные цитаты из предопределённого списка;
* просматривать историю сгенерированных цитат (последние 10);
* фильтровать цитаты по автору и теме;
* сохранять историю и данные между запусками в формате JSON.

## Технологии

* Python 3
* Tkinter (GUI)
* JSON (хранение данных)
* Git (контроль версий)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/zolsvetl17-lab/IAKertbieva.git
2. Перейдите в папку проекта:
   ```bash
   cd random-quote-generator
3. Запустите приложение:
   ```bash
   python main.py
### Использование
1. Нажмите кнопку **«Сгенерировать цитату»**, чтобы увидеть случайную цитату.
2. Используйте выпадающие списки для фильтрации цитат по **автору** или **теме**.
3. Нажмите **«Очистить историю»**, чтобы сбросить историю сгенерированных цитат.
4. Нажмите **«Сохранить данные»**, чтобы сохранить текущие цитаты и историю в quotes_data.json.

## Примеры использования (тесты)
### **Тест 1: Генерация цитаты**
1. Запустите приложение.
2. Нажмите «Сгенерировать цитату».
3. Убедитесь, что цитата отображается в интерфейсе.
4. Повторите несколько раз — цитаты должны быть разными.

### **Тест 2: Фильтрация**
1. Выберите автора из выпадающего списка.
2. Нажмите «Сгенерировать цитату» — должна появиться цитата выбранного автора.
3. Аналогично проверьте фильтрацию по теме.

### **Тест 3: Сохранение и загрузка**
1. Сгенерируйте несколько цитат.
2. Закройте приложение.
3. Перезапустите приложение — история должна восстановиться.

### **Тест 4: Очистка истории**
1. Сгенерируйте цитаты.
2. Нажмите «Очистить историю» — история должна стать пустой.
